================================================================================
  Lobster 在线版
================================================================================

  1. 解压到本程序文件夹，全部覆盖。
  2. 双击「一键启动.bat」（首次会安装依赖，等结束）。
  3. 浏览器打开：http://127.0.0.1:8000

  素材剪辑叠字需 ffmpeg：完整包可含 deps\ffmpeg\ffmpeg.exe（制作者打代码包时设
  INCLUDE_FFMPEG_IN_PACK=1）；否则请自行安装 ffmpeg 或设 LOBSTER_FFMPEG_PATH。
