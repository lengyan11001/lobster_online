推荐使用流程

配置中心登录账号并保存规则 -> 搜索采集关键词视频或监控同行 -> 视频评论抓取客户 -> 查看全部客户和精准客户 -> 对精准客户做关注评论或私信互动 -> 需要站外线索时使用陌生人留资、提及评论或群成员提取 -> 遇到异常时查看执行日志。

这类在线手册怎么更新

页面内容是静态 HTML，适合直接随网站一起部署。后续如果抖音工作台改版，只需要重新生成对应截图，并调整本页面的步骤和注意事项。用户访问 `/static/douyin-origin/douyin.html-guide` 时会看到最新版本，不需要单独下载文档。

本手册只说明系统页面的使用方法。实际执行效果会受到账号状态、平台页面变化、网络环境、关键词质量和配置规则影响。批量执行前建议先用少量任务测试。
