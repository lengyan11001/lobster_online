# install.bat 与完整包（以仓库根目录 `install.bat` 为准）

**唯一权威**：`lobster_online/install.bat`（与制包/交付约定一致）。  
下文描述其行为；若与脚本不一致，**以 `install.bat` 为准**。

---

## 默认行为（不设 `LOBSTER_OFFLINE_ONLY`）

目标：**尽量用包内离线资源**；若某步失败，**允许联网补救**（不是「为离线而离线」）。

- **pip**：无 pip 时依次尝试：`pip_bootstrap_from_wheel` / wheel 引导 → `deps\get-pip.py` → `ensurepip`（后者可能需网）。
- **VC++**：优先包内 `deps\vc_redist.x64.exe`；无则 **联网下载** 静默安装（可 `LOBSTER_SKIP_VCREDIST=1` 跳过）。
- **Python 依赖**：先 `deps\wheels` + `--no-index`；失败可 **在线** `pip install -r`（若 `requirements.txt` 含 `tos`，主步骤会生成 `requirements.runtime.txt` 跳过 `tos`，**2b** 再装 tos）。
- **2b**：pycryptodome / tos 优先离线 wheel，否则可在线安装；失败时 **SKIP** 部分能力（WeCom / TOS），不强制退出。
- **OpenClaw**：未预装时可 **npm install openclaw**（除非 `LOBSTER_OFFLINE_ONLY=1`）。
- **Playwright**：无 `browser_chromium` 时可 **playwright install chromium**（除非 `LOBSTER_OFFLINE_ONLY=1`）。
- **ffmpeg**：无 `deps\ffmpeg\ffmpeg.exe` 时可按脚本下载（除非 `LOBSTER_OFFLINE_ONLY=1`）。

---

## 强制纯离线（无网或必须禁止联网时）

安装前在 **cmd** 中执行：

```bat
set LOBSTER_OFFLINE_ONLY=1
install.bat
```

在此模式下 **不** 走在线 pip / npm / playwright 下载 / ffmpeg 下载等；须自备齐全离线文件，否则按脚本报错或 **SKIP**。

可选：

| 变量 | 作用 |
|------|------|
| `LOBSTER_SKIP_VCREDIST=1` | 跳过 VC++ 检测与安装（需本机已装 VC++ 2015–2022 x64） |
| `LOBSTER_SKIP_DESKTOP_SHORTCUT=1` | 跳过桌面快捷方式 |
| `LOBSTER_SKIP_INSTALL_PAUSE` | 安装结束不 pause |

## 制包机：打「全依赖」zip

**约定**：`pack_full.sh` / `pack_full_project.sh` **只打 zip**，**不修改** `install.bat`、`start.bat`、`run_backend.bat`、`run_mcp.bat`。依赖只放在 `lobster_online` 目录内（`deps/`、`python/`、`nodejs/` 等）。

**生产上线、干净包（排除 `docs/`、`pack_bundle.env`、根目录说明性 `.md` 等）**：见 **`docs/生产打包流程.md`**。

| 步骤 | 命令 |
|------|------|
| 只看缺口、不下载 | `python3 scripts/report_pack_gaps.py` |
| 只拉齐依赖、不打包 | `bash scripts/sync_deps_for_pack.sh` |
| 检查 → 拉齐 → 复检 → zip | `bash scripts/build_result_package.sh` |

`build_result_package.sh` 会依次：`report_pack_gaps`（仅展示）→ `build_package.sh` → `deps/vc_redist.x64.exe` → `ensure_full_pack_deps.sh` → `browser_chromium`（必要时）→ **`report_pack_gaps`（复检，核心不齐则中止）** → `pack_full_project.sh`。

依赖**已经齐**、只想打 zip：

```bash
SKIP_ENSURE_FULL_PACK_DEPS=1 bash pack_full_project.sh
# 或带「使用说明.txt」打入 zip 的旧版完整包：
SKIP_ENSURE_FULL_PACK_DEPS=1 bash pack_full.sh
```

**用户说明**：根目录 **`使用说明-完整包.txt`**（仅 **`pack_full.sh`** 会临时复制为 `使用说明.txt` 打入 zip；**`pack_full_project.sh`** 不打入该说明，适合纯交付）。

## 建议打入完整包的内容

| 路径 | 说明 |
|------|------|
| `deps/wheels/` | `pip-*.whl` 及 `requirements.txt` 对应 wheel |
| `deps/vc_redist.x64.exe` | 可选；无则 install 内联下载（需网） |
| `browser_chromium/` | 离线 Playwright |
| `deps/ffmpeg/ffmpeg.exe` | 剪辑；无且非离线模式可由脚本下载 |
| `nodejs/node_modules/openclaw/` | 离线对话网关；否则在线 `npm install` |

强制重新下载全部 wheel：`FORCE_PREPARE_OFFLINE=1`（见 `ensure_full_pack_deps.sh` / `build_result_package.sh`）。
