# 文档与规则索引（防断片）

本页汇总 **Cursor 规则（`.mdc`）** 与 **本仓库核心 `docs/`**，改代码前先扫一眼，避免与架构/鉴权/部署约定冲突。

---

## 一、`.cursor/rules/*.mdc`（工作区根，共 6 个）

| 文件 | 要点 |
|------|------|
| **`only-online-and-server.mdc`** | 可改 `lobster_online/`、`lobster-server/` 等；**无**独立 `lobster/` 目录；未指定时**默认优先 `lobster_online/`**。不要把 `lobster_online` 整目录 scp 到 ECS。 |
| **`server-ssh-operations.mdc`** | 云端 API：`lobster-server` git push → **`bash scripts/deploy_from_local.sh`**。`lobster_online/` **不上服务器**；本机释放 8000 后 `./start_online.sh`。 |
| **`auto-update-restart-after-changes.mdc`** | 改完代码：本地在线版按 `docs/本地启动与重启.md` 重启后端。 |
| **`api-auth-required.mdc`** | 在 **`lobster-server/`** 新增路由默认须鉴权；例外要写明。见 `lobster-server/docs/接口鉴权规范.md`。 |
| **`online-chat-openclaw-local.mdc`** | **`/chat`、`/chat/stream` 必须打本机 `LOCAL_API_BASE`**，勿默认远端 `API_BASE`；OpenClaw 只在本机。 |
| **`no-fallback.mdc`** | **禁止**静默兜底/自动降级链；不满足就报错退出。 |

**易错（架构）**：**`media.edit`（素材剪辑）只在 `lobster_online` 本机跑**；**`lobster-server` 只做鉴权/积分等**，不要把 `media_edit` 打进 ECS API 仓库。
